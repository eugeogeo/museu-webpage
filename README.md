# Museu we-page

## Sobre o projeto 

Projeto de extensão da UTFPR - Campus Campo Mourão em parceria com o Museu Municipal Deolindo Mendes Pereira.

## Como rodar
 ```
   git clone https://github.com/eugeogeo/museu-webpage

   cd museu-webpage

   yarn 

   yarn dev

 ```
